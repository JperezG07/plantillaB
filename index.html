HTML

Es el lenguaje estándar utilizado para crear y diseñar páginas web. Consiste en una serie de elementos o "etiquetas" que estructuran el contenido, como texto, imágenes, enlaces y otros elementos multimedia.

CSS

Es un lenguaje utilizado para describir la presentación y el diseño de un documento HTML. Mientras que HTML se encarga de la estructura del contenido, CSS permite aplicar estilos como colores, fuentes, márgenes, espaciado y disposición de los elementos en la página.

JS (Java Script)

Es un lenguaje de programación que se utiliza principalmente para crear interactividad y dinámicas en las páginas web. Es un componente esencial del desarrollo web moderno, junto con HTML y CSS.
